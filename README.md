# KOB Platform

## Setup (one time)
1. Open Supabase → SQL Editor → paste schema.sql → Run
2. Drag this entire folder to netlify.com/drop
3. Go to your-netlify-url/auth.html and create your account

## Using it
- Dashboard: upload any KOB viewer HTML file to create an event
- Copy the share link and send to players — they see live scores
- You enter scores from any device using the same link (after signing in)

## Updating the viewer
When a new viewer version is available, download the updated kob-saas.zip
from Claude and re-drag to Netlify Drop. Your events and scores are safe in
Supabase — only the viewer code updates.
